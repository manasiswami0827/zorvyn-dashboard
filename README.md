#  Finance Dashboard

A modern and responsive **Finance Dashboard** built using React and Vite. This application helps users track transactions, visualize financial data, and manage expenses efficiently with a clean UI and role-based access.

---

## 🚀 Live Demo

👉https://zorvyn-dashboard-xi.vercel.app

---

## 📌 Features

* 🔐 **Role-Based Access**

  * Admin: Add transactions
  * Viewer: Read-only access

* 💳 **Transaction Management**

  * Add new transactions
  * Categorize income and expenses
  * View transaction history

* 📊 **Data Visualization**

  * Time-based financial trends
  * Category-wise expense breakdown

* 📋 **Dashboard Overview**

  * Summary cards for balance, income, and expenses

* 📱 **Responsive Design**

  * Works across desktop, tablet, and mobile devices

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite
* **Styling:** CSS (Modular + Responsive Design)
* **Icons:** Lucide React
* **State Management:** React Hooks
* **Deployment:** Vercel
* **Containerization:** Docker

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/manasiswami0827/zorvyn-dashboard.git
cd zorvyn-dashboard
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

---

## 🐳 Docker Setup (Optional)

Build and run using Docker:

```bash
docker build -t finance-dashboard .
docker run -p 5173:5173 finance-dashboard
```
