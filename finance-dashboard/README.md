# 💰 Finance Dashboard

A modern and responsive Finance Dashboard built using React and Vite.  
It helps users track income, expenses, and financial insights with a clean UI.

---

## 🚀 Features

- 📊 Dashboard with summary cards
- 📈 Charts for financial insights
- 💸 Add transactions (Admin only)
- 🔍 Transaction list with details
- 🔐 Role-based UI (Admin / Viewer)
- 🌙 Dark mode support
- 📱 Fully responsive design
- 🐳 Docker support

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** CSS (Custom, Responsive Design)
- **Icons:** Lucide React
- **State Management:** React Hooks
- **Containerization:** Docker
